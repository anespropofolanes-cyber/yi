# 中醫小白的新手村

一個用「倪海廈老師的思想」帶完全沒基礎的人（中醫小白）入門中醫的互動網頁。
不是百科，是一座可以「邊玩邊學」的學堂：看病人、選問題、辨藥材、摸脈象、下判斷，由一位「師父」一路帶著走。

> **重要免責聲明**
> 本站為中醫入門與倪海廈課程思想之**學習整理**，**不是診斷、不是處方，也不能拿來自我治療**。
> 所有方藥、劑量、穴位僅供認識與理解；附子、麻黃、峻下等高風險內容只供觀摩。
> 任何用藥、針灸都要由執業中醫師面診決定；出現各病的「紅線症狀」請立即就醫。

## 怎麼開啟

這是一個**單一檔案的純前端網頁**，不需要安裝、不需要伺服器、不用建置。

- 本機看：用瀏覽器直接打開 `index.html` 即可。
- 圖片要顯示，`index.html` 旁邊必須有 `assets/` 資料夾（裡面是全部圖片）。

## 部署到 GitHub Pages

倉庫只要兩樣東西：`index.html` 和整個 `assets/` 資料夾。

```
（倉庫根目錄）
├── index.html      ← 主程式（單檔）
└── assets/         ← 全部圖片（21 張 .png）
```

步驟：

1. 建一個 repo，把 `index.html` 與 `assets/` 資料夾一起上傳（圖片要整夾，保留 `assets/` 路徑）。
2. Settings → Pages → Source 選「Deploy from a branch」→ 分支 `main`、資料夾 `/ (root)` → Save。
3. 等一兩分鐘，網址為 `https://你的帳號.github.io/repo名稱/`。

注意事項：

- **圖片檔名大小寫**要與程式裡完全一致（GitHub 區分大小寫，本機 Mac 不分，最常在這裡踩雷）。
- 程式**沒有任何外部 CDN 或 JavaScript 套件**依賴，純 HTML/CSS/JS，離線也能跑；文中的 PubMed／Wiley 連結只是參考文獻外連，點了才開。
- 學習進度與「我的筆記」存在瀏覽器 `localStorage`，**換裝置或清除瀏覽資料不會同步**，這是正常的。

## 內容總覽

學習地圖目前有 18+ 關，循序漸進：

- **入門**：中醫概論、互動診間、選路線
- **基礎理論**：身體像一座城市、陰陽、五行相生相剋、氣血津液
- **四診**：望聞問切、舌診入門、脈診入門
- **辨證**：八綱辨證、六經辨證
- **心法**：倪海廈九大思維模型（這座學堂的核心）
- **藥方針**：藥材圖鑑、方劑圖鑑（含古方參考用量）、藥怎麼變成方（君臣佐使／七情配伍）、針灸入門、查找（藥·方·穴）
- **實戰**：病例推理、跟診實戰 A01–A04、倪師醫案（進階觀摩）
- **驗收與生活**：隨堂考綜合複習、日常養生、常見問題證據速覽、我的筆記

每個診斷／養生模組都附一段「**誠實版**的現代研究」：照可信結論校準，不誇大、標證據等級、講清楚就醫紅線。

- 病例 23｜跟診實戰 4｜倪師醫案 24（觀摩）
